# ESP as Universal Tool

## Idea
Use ESP32 as a universal platform for multiple experimental systems.

## Why
- Familiar platform
- Fast prototyping
- Cheap and flexible

## Possible Uses
- Control interface (buttons, web UI)
- Sensor hub
- Communication bridge
- Audio processing experiments

## Insight
ESP32 is repeatedly used as a default solution ("ban hammer").

## Risk
Overusing it even when better tools exist.
