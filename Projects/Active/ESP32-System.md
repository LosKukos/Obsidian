# ESP32 System

## Description
Personal experimental platform based on ESP32 combining multiple modules.

## Stack
- FreeRTOS
- Async Web Server
- OLED display
- WS2812 LED

## Goals
- Combine multiple features into one system
- Maintain responsiveness (UI + network)

## Known Issues
- Conflicts between async server and display tasks
- Task synchronization complexity

## Notes
Project is exploratory and iterative.
